# `hs-bindgen`

`hs-bindgen` is a [Haskell][] library that *automatically* generates Haskell FFI
bindings from C header files.

See the [repository README][] for the manual and other information.

[Haskell]: https://www.haskell.org/
[repository README]: https://github.com/well-typed/hs-bindgen#hs-bindgen
