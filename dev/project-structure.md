# Project Structure

This document provides an overview of the `hs-bindgen` project structure to
help new developers understand how the codebase is organized.

## Repository Layout

```
hs-bindgen/
├── alternatives/
├── ansi-diff/
├── c-expr/
├── clang/
├── dev/
├── hs-bindgen/
├── hs-bindgen-runtime/
├── hs-bindgen-test-runtime/
├── manual/
├── scripts/
└── userland-capi/
```
