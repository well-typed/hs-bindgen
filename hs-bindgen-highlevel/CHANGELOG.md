# Revision history for hs-bindgen-highlevel

## 0.1.0 -- YYYY-mm-dd

* Initial release. A combinator library for hand-writing high-level, type-safe
  Haskell wrappers over the low-level FFI bindings generated by hs-bindgen. These
  modules were previously part of `hs-bindgen-runtime` under
  `HsBindgen.Runtime.HighLevel.*`; they now live here under `HsBindgen.HighLevel.*`.

  The vocabulary: a `Marshal` arrow moves a Haskell value into the C argument(s) it
  fills (`scalar`, `bracket`, `at`, `>>>`, `marshalOptional`, `asConstArg` /
  `asMutableArg`); `MarshalStruct` / `UnmarshalStruct` write and read whole structs;
  `Unmarshaller` handles out-parameters. `HighLevel` lifts a low-level callable into
  a wrapper with `input` / `output` / `scratch` / `fixed` and a closer (`resultPure`,
  `resultIO`, `discardResult`, and the error-aware `throwOn` / `throwOnNonZero` /
  `throwOnOut`), drops a struct marshaller into a position with `asArgument` /
  `asOutput` / `asResult`, and collapses a void closer's trailing `()` with
  `dropTrailingUnit`. `Defaults` provides per-type defaults and `auto`, which fills the
  mundane positions from the wrapper's signature. `assertPure` exposes a deterministic
  call as a pure function, and `HighLevel.Unlifted` handles small by-value structs
  GHC's FFI cannot pass directly. See the README for a guide.
